## Next Steps

<Steps>
  <Step title="Install TypeDoc Mintlify">
    Install the plugin in your TypeScript project
    ```bash
    npm install typedoc-mintlify typedoc --save-dev
    ```
  </Step>
  <Step title="Configure TypeDoc">
    Create a `typedoc.json` configuration file
    ```json
    {
      "entryPoints": ["src/index.ts"],
      "out": "docs",
      "plugin": ["typedoc-mintlify"],
      "theme": "mintlify"
    }
    ```
  </Step>
  <Step title="Generate Documentation">
    Run TypeDoc to generate your documentation
    ```bash
    npx typedoc
    ```
  </Step>
  <Step title="Preview with Mintlify">
    Start the Mintlify dev server
    ```bash
    npx mintlify dev
    ```
  </Step>
</Steps>

  You're now ready to explore the example API documentation!
