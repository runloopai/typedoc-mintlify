# typedoc-mintlify Example

This example demonstrates how to use typedoc-mintlify to generate Mintlify-compatible documentation from TypeScript code.

## Quick Start

1. **Install dependencies**:

   ```bash
   npm install
   ```

2. **Generate API documentation**:

   ```bash
   npm run docs
   ```

3. **Preview with Mintlify**:

   ```bash
   npm run preview
   ```

4. **Or do both at once**:
   ```bash
   npm start
   ```

Your documentation will be available at `http://localhost:3000` ðŸŽ‰

## What's Included

This example includes:

### Source Code

- **`src/index.ts`**: Example TypeScript code with comprehensive JSDoc comments

### Configuration

- **`typedoc.json`**: TypeDoc configuration using the Mintlify plugin
- **`tsconfig.json`**: TypeScript compiler configuration
- **`mint.json`**: Mintlify configuration with navigation

### Documentation Pages

- **`introduction.mdx`**: Welcome page with overview
- **`installation.mdx`**: Installation instructions
- **`quickstart.mdx`**: Quick start guide
- **`api/`**: API reference pages (generated by TypeDoc)

The example TypeScript file demonstrates:

- Interfaces with detailed documentation
- Functions with parameters and examples
- Classes with methods and properties
- Enums and type aliases
- Error handling patterns

## Configuration

The `typedoc.json` file shows how to configure typedoc-mintlify:

```json
{
  "$schema": "https://typedoc.org/schema.json",
  "entryPoints": ["src/index.ts"],
  "out": "api",
  "plugin": ["typedoc-mintlify"],
  "theme": "mintlify",
  "mintlifyComponents": true
}
```

## Available Scripts

```bash
# Generate API documentation from TypeScript
npm run docs

# Watch TypeScript files and regenerate on changes
npm run docs:watch

# Preview documentation with Mintlify dev server
npm run preview

# Generate docs and start preview server
npm start
```

## Generated Output

After running `npm run docs`, the `api/` directory will contain:

- **Markdown files** for each class, interface, function, and type
- **Proper frontmatter** on each page with titles and descriptions
- **Mintlify components** like `<ParamField>`, `<Warning>`, `<Note>`, etc.
- **Type information** with proper formatting
- **Code examples** from @example tags

## Next Steps

1. **Explore the docs**: Visit `http://localhost:3000` after running `npm start`

2. **Modify the code**: Edit `src/index.ts` and regenerate to see changes

3. **Customize**: Update `mint.json` to change colors, navigation, or branding

4. **Add pages**: Create new `.mdx` files alongside the generated API docs
