<Info>
This class provides a high-level interface for user management operations.
It handles caching, error handling, and retry logic automatically.
</Info>

- [Methods](#methods)

### Constructor

`new constructor(apiKey: string, baseUrl: string)`

Creates a new UserService instance

<Expandable title="Parameters">
<ParamField header="apiKey" type="string" required>
API key for authentication
</ParamField>

<ParamField path="baseUrl" type="string" required>
Base URL for the API (defaults to production)
</ParamField>
</Expandable>


### Methods

<Expandable title="Parameters">
<ParamField path="userId" type="string" required>

</ParamField>
</Expandable>

<Expandable title="Returns">
<ResponseField name="returns" type="Promise<User>">

</ResponseField>
</Expandable>
